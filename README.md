## Português

Este é o meu primeiro projeto prático estudando **HTML5**. Explorei diferentes tipos de `inputs` e a importância da acessibilidade.

### O que eu aprendi:
* **Tag `<label>`**: Associação com `id` via atributo `for`.
* **Tipos de Input**: `text`, `email`, `password`, `radio`, `checkbox`, `date`, `file` e `select`.
* **Validação**: Uso do atributo `required`.

### Tecnologias:
* HTML5.
